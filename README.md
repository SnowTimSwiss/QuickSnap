# SnapArea

SnapArea is a minimal screenshot tool for Linux.

Its goal is simple:

Press a shortcut.

Drag a rectangle.

The screenshot is immediately copied to the clipboard.

That's it.

No editor.
No floating toolbar.
No save dialog.
No unnecessary clicks.

## Features

- Instant area selection
- Direct clipboard copy
- Keyboard shortcut friendly
- Lightweight
- Native Wayland support
- X11 support (planned)
- Open source

## Philosophy

Modern screenshot tools have become image editors.

SnapArea focuses on doing one thing well:

Capture a selected screen region as fast as possible.
